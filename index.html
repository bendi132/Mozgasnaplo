🔑 Google Cloud beállítás
⚠️ Ezt egyszer kell elvégezni. Utána az app automatikusan elmenti az adataidat a saját Google Sheets táblázatodba.
1
Google Cloud Projekt
Menj a console.cloud.google.com oldalra, és hozz létre egy új projektet (pl. „MozgásNapló").
2
Google Sheets API engedélyezése
APIs & Services → Library → „Google Sheets API" keresés → Enable gomb.
3
OAuth 2.0 Client ID létrehozása
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application típus.
Az Authorized JavaScript origins mezőbe írd be a fájl helyét, pl.: http://localhost:8080 vagy azt az URL-t ahol megnyitod.
Másold ki a kapott Client ID-t az alábbi mezőbe.
4
Google Sheets táblázat ID
Hozz létre egy üres Google Sheets táblázatot. Az URL-ből vedd ki az azonosítót:
docs.google.com/spreadsheets/d/[EZ AZ ID]/edit
